## 🎯 How It Works - Interview Demo

1. **Enter LeetCode Username** → Click `FETCH`
2. **First Load:** `API CALL 400ms` - Fetches from LeetCode GraphQL
3. **Second Load:** `CACHE HIT <20ms` - Served from LRU cache
4. **4th User:** LRU Eviction - Oldest user removed from cache
5. **L60 Score:** Calculated as `(Easy*1 + Medium*3 + Hard*5) / 10`
